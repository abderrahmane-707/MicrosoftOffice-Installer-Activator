# Description
Download and install Microsoft Office (2016 / 2019 / 2021 / 365)  
And activate them

# How To Use
1. Disable Windows Defender or any antivirus software Before run script
2. Right-click on `Office.bat` and select **Run as administrator**
3. Make sure you are connected to the Internet
4. Choose the Office programs you want to download
5. You can choose the version of the package,language,mode of Installation
6. Enjoy!
